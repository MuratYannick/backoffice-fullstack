# 🚀 Erosion des Ames - stage 1

## Architecture
- **Frontend**: React + Vite + Tailwind CSS (port 5173)
- **Backend**: Express + Node.js (port 3000)
- **Database**: MySQL (à venir)

## Installation et démarrage
### mock-up
cd mock-up
npm install
npm run dev

### Frontend
cd EDA/frontend
npm install
npm run dev

### Backend
cd EDA/backend
npm install
npm run dev
